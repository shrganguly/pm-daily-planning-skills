## Keyword Shorthand

**Instead of:**
"@Shreya in #product-team from John asking about PRD review deadline"

**Type:**
"@product John PRD deadline"

**I'll expand it in the daily plan to:**
- [ ] **#product-team** - @Shreya PRD review deadline (John)

---

## Common Patterns

**@Mentions:**
- Format: `@channel person topic`
- Example: `@product John PRD`, `@design Sarah mockups`

**Direct Messages:**
- Format: `DM person topic`
- Example: `DM Sarah roadmap`, `DM Mike question`

**Channel Messages:**
- Format: `#channel topic`
- Example: `#eng blocker`, `#product feature-request`

---

## Pro Tips

1. **Use abbreviations:**
   - "prod" = product-team
   - "eng" = engineering
   - "des" = design

2. **Skip if no action needed:**
   - Only capture messages that need YOUR response
   - FYI messages â†’ skip

3. **Batch similar items:**
   - "3 DMs about roadmap" instead of listing each

4. **Use urgency markers:**
   - "!" = urgent
   - Example: `@product John PRD!` = needs response today
